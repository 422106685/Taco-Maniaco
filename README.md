# Taco-Maniaco
Proyecto 3 desarrollo web
La realización de Proyecto 3 de desarrollo web, nos enseñó a manejar herramientas como HTML, JavaScript y CSS.

¿Cómo funciona el proyecto?
  El proyecto presenta imagenes de las diferentes opciones de tacos, en la parte superior se desplega una nuve con una breve descripción, el usuario deberá elejir la opción que mas le interese y deberá de seleccionar agregar a carrito, el carrito hace el calculo del producto mas el IVA y nos muestra el total y el adicional (IVA) agregado.
  
¿Qué funcionalidades implementaron?
renderCatalog(): nos ayuda a dibujar los productos.

  addToCart(productId): Proceso de compra, busca el producto, verifica duplicados, nuevos productos y actualiza.

  removeFromCart(productId): Crea un nuevo carrito que incluya todos los productos excepto el que tenga este ID". Es una forma limpia de borrar un elemento de la lista.

  updateCartUI():  hace los cálculos finales, bucle de cálculo que ecorre el carrito y multiplica precio * cantidad para obtener el subtotal.
Cálculo de IVA, Aplica el 16% sobre el subtotal (subtotal x 0.16).

  En el código HTML, JavaScript y CSS se presentan a mayor detalle las descripciones.

¿Qué retos encontraron?
